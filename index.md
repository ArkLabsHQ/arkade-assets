# ArkAsset Protocol Documentation

Welcome to the official documentation for the ArkAsset Protocol. This repository contains the technical specifications, API references, and examples needed to understand and build with ArkAssets.

## Core Concepts

- **ArkAssets (`arkassets.md`)**: The core specification for the ArkAsset protocol, detailing its structure, transaction formats, and rules for issuance, transfer, and control.
- **Arkade Script (`arkade-script.md`)**: A guide to the scripting capabilities, including the introspection opcodes available for interacting with ArkAssets in smart contracts.

## Examples

- **Transaction & Contract Examples (`examples.md`)**: A comprehensive collection of examples, including detailed transaction structures and sample Arkade Script contracts for various use cases.
- **ArkadeKitties (`ArkadeKitties.md`)**: A fun, practical example of a simple NFT project built using ArkAssets.

---

### Getting Started

To get started, we recommend reading the `arkassets.md` file to understand the fundamental concepts of the protocol. After that, you can explore the `examples.md` to see how these concepts are applied in practice.
