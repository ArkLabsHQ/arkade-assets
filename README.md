- **[Arkade Script](arkade-script.md)**: A guide to the scripting capabilities, including the introspection opcodes available for interacting with Arkade Assets in smart contracts.

## Examples

- **[Transaction & Contract Examples](examples.md)**: A comprehensive collection of examples, including detailed transaction structures and sample Arkade Script contracts for various use cases.
- **[ArkadeKitties](ArkadeKitties.md)**: A fun, practical example of a simple NFT project built using Arkade Assets.
